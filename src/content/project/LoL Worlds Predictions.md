#### 簡介
本專案旨在使用機器學習模型預測英雄聯盟 (LoL) 世界錦標賽的比賽結果。本專案包含兩個主要元件：一個用來擷取比賽資料的網路爬蟲，以及一個用來預測比賽結果的機器學習模型。

#### 使用技術
- Python
- Scikit-learn
- Pytorch
- pandas
